# 🔐 Password Analyzer

A Python-based Password Analyzer that evaluates the strength of a password and provides suggestions to improve its security.

## 📌 Project Overview

This project analyzes passwords based on common security rules and classifies them as:
- 🟢 Strong
- 🟡 Medium
- 🔴 Weak

It also provides recommendations to help users create stronger and more secure passwords.

## ✨ Features

- Checks password length
- Checks for uppercase letters
- Checks for lowercase letters
- Checks for numbers
- Checks for special characters
- Displays password strength
- Provides improvement suggestions

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Gayatri6619/Password_Analyzer.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Password_Analyzer
   ```

3. Run the Python program:
   ```bash
   python password_analyzer.py
   ```

4. Enter a password when prompted.

5. The program will analyze the password and display:
   - Password Strength (Weak/Medium/Strong)
   - Suggestions to improve password security (if needed)
