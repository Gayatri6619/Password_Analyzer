# Password Analyzer

## Project Description
This is a Python-based Password Analyzer that checks the strength of a password.

## Features
- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Displays password strength (Weak, Medium, Strong)
- Suggests how to improve weak passwords

## Technologies Used
- Python 3
- Regular Expressions (re)

## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:
   python password_analyzer.py
